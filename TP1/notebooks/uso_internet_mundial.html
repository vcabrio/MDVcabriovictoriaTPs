Luego de leer los archivos CSV y ver como es la tabla gracias al “head” normalizamos los nombres de las columnas a traves de una funcion:

normalizacion = function(uso_internet, clean = TRUE, which = "cols") {

  if (clean) {
    data_clean = janitor::clean_names(uso_internet)
  } else {
    data_clean = uso_internet
  }

  if (which == "both") {
    data_clean = data_clean |> 
      janitor::remove_empty(which = "cols") |> 
      janitor::remove_empty(which = "rows")
  } else {
    data_clean = janitor::remove_empty(data_clean, which = which)
  }

  return(data_clean)
}

internet_clean = normalizacion(uso_internet, which = "both")
paises_clean = normalizacion(paises, which = "both")

Al encontrar los valores faltantes (462) los eliminamos de la base para que no modifiquen nuestro promedio a partir de la funcion drop_na:

long_internet <- long_internet %>%

Una vez que tenemos limpia nuestra base sobre el uso de internet, hacemos un left join para matchear las tablas long_internet y piases_clean:

tablas_joined <- long_internet %>%
  left_join(paises_clean, by = "country_code")

Para comprobar la hipotesis debemos hacer el promedio de ingresos por region, agrupando por region y año: Para esto utilizamos las funciones de summarize y group by.

tabla_ingresos <- tablas_joined %>%
group_by(income_group, year, region, indicator_name) %>%
summarise(
prom = mean(usu_internet, na.rm = TRUE),
.groups = "drop"
)

Convertimos en factores los caracteres para poder asignarle un valor de forma jerarquica a internet join y no de forma alfabetica. Luego con la funcion arrange lo que hacemos es ordenarlo definitivamente de forma jeraerquica, de menor a mayor.

tablas_joined$income_group = factor(
  tablas_joined$income_group,
  levels = c(
    "Low income",
    "Lower middle income",
    "Upper middle income",
    "High income"
  )
)
tablas_joined = tablas_joined %>%
  arrange(income_group, year)

Ahora armamos la tabla final para poder comprobar la hipotesis:

resultado_hipotesis = 
  
  tablas_joined %>% group_by(year, income_group, indicator_name) %>%


summarize(
usu_internet = mean(usu_internet, na.rm = TRUE)
) %>%


pivot_wider(
names_from = income_group,
values_from = usu_internet
) %>%

select(
year,
`Low income`,
`Lower middle income`,
`Upper middle income`,
`High income`,
indicator_name
)

using the `.groups` argument.
